# Sequential Thinking Skill

복잡한 문제를 단계별로 분석하는 구조화된 사고 프로세스를 실행합니다.

## 분석 프레임워크

### 1단계: 문제 분해
- 핵심 질문 명확화
- 구성 요소 식별 (3+ 컴포넌트)
- 의존성 매핑

### 2단계: 가설 수립
- 가능한 원인/해결책 나열
- 각 가설의 검증 방법 정의
- 우선순위 결정

### 3단계: 증거 수집
- 코드/로그/문서 조사
- 패턴 식별
- 모순점 확인

### 4단계: 분석 및 추론
- 증거 기반 결론 도출
- 대안 평가
- 리스크 식별

### 5단계: 종합 및 권장사항
- 핵심 발견 요약
- 구체적 행동 방안
- 후속 검증 계획

## 적용 상황
- 복잡한 디버깅 (다중 레이어)
- 아키텍처 분석/설계
- 성능 병목 조사
- 보안 취약점 분석
- 루트 원인 분석

## 출력 형식

```
🎯 문제 정의: [핵심 질문]

📊 분석 단계:
1. [구성 요소 분석]
2. [가설 및 검증]
3. [증거 및 발견]
4. [추론 및 결론]

💡 권장사항:
- [구체적 행동 1]
- [구체적 행동 2]

⚠️ 주의사항/리스크:
- [식별된 리스크]
```

---

위 프레임워크를 사용하여 사용자의 질문/문제를 체계적으로 분석하세요.
