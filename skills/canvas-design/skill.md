# Canvas Design

Creates museum-quality visual art in PNG and PDF formats by establishing design philosophies that are then expressed visually.

## Trigger
- When user needs professional visual design
- Creating gallery-worthy artwork
- Designing visual compositions
- Building brand visual identity

## Core Process

1. **Design Philosophy Creation** (.md)
   - Develop aesthetic movement manifesto (4-6 paragraphs)
   - Emphasize visual expression over text
   - Conceptual depth and sophistication

2. **Canvas Expression** (.pdf/.png)
   - Translate philosophy into visual composition
   - ~90% visual, ~10% essential text
   - Museum/magazine quality

## Key Principles

**Authenticity:**
- Create original designs, never copy existing artists' work
- Avoid copyright violations

**Execution:**
- Meticulously crafted work suggesting master-level expertise
- Perfect spacing, no overlaps, flawless formatting
- Systematic visual language from careful observation

**Design Elements:**
- Limited, intentional color palette
- All elements contained within canvas boundaries
- Proper margins and breathing room
- Design-forward typography using thin fonts

**Content Balance:**
- ~90% visual composition
- ~10% essential typography
- Typography as visual element, not decoration
- Subtle conceptual references woven invisibly into form and color

## Output Quality

- Museum/magazine worthy
- Professional visual language
- Gallery-ready compositions
- Sophisticated artistic presentation

## Use When

- Creating professional artwork
- Designing visual identities
- Building gallery-quality compositions
- Expressing design philosophies visually
- Creating artistic visual content
