# Brand Guidelines (Anthropic)

Applies Anthropic's official visual identity to artifacts needing corporate branding.

## Trigger
- When artifact needs Anthropic branding
- Corporate identity and styling required
- Post-processing designs with brand colors
- Applying consistent visual identity

## Brand Elements

**Color Palette:**
- Primary Dark: `#141413`
- Primary Light: `#faf9f5`
- Accent Orange: `#d97757`
- Accent Blue: `#6a9bcc`
- Accent Green: `#788c5d`

**Typography:**
- Headings: Poppins font (24pt+)
- Body text: Lora font
- Fallbacks: Arial/Georgia if custom fonts unavailable

## Primary Use Cases

- Branding and corporate identity
- Visual identity and styling
- Post-processing designs
- Consistent brand application
- Design system implementation

## Technical Approach

**Font Management:**
- Uses system-installed typefaces
- Built-in fallbacks for cross-platform compatibility

**Color Application:**
- Maintains brand fidelity with precise RGB values
- Cycles through accent colors for visual hierarchy
- Preserves on-brand aesthetics

## Use When

- Creating Anthropic-branded materials
- Applying consistent visual identity
- Designing corporate presentations
- Branding design artifacts
- Ensuring brand consistency across designs

## Complementary Skills

Works with:
- artifacts-builder for Anthropic-branded web interfaces
- canvas-design for branded visual compositions
